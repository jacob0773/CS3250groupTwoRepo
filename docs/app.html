sticky note app.

a simple sticky note app built for students.

write and organize notes without extra clutter.

built in Java.

features

  • quick notes - open the app and start writing
  • easy organization - keep your notes sorted and grouped
  • customizable - change fonts, colors, and layout
  • search - find notes faster
  • lightweight - runs locally on your machine

about

the sticky note app is a desktop Java application for students who want

a simple way to write and organize notes. no accounts, no cloud,

and no extra setup. just open it and start writing.

it is a simple sticky note app built for our class project.
